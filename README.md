# 📘 Claude101

> **Stop treating AI like Google. Start thinking with it.**

Claude101 is a beginner-to-advanced guide that helps you understand how to use Claude as a **thinking partner**, not just a search tool.

---

## 🚀 About The Project

Most people use AI like a search engine — asking quick questions and expecting quick answers.

But Claude is designed for something more powerful:

* reasoning
* structured thinking
* collaboration

**Claude101** is built to shift that mindset.

This project teaches users how to:

* move beyond basic prompts
* interact with AI more effectively
* unlock deeper, more useful outputs

---

## 🎯 Goal

To help users transition from:

> ❌ “Ask → Get Answer”

to

> ✅ “Think → Collaborate → Build”

---

## 🧠 What You’ll Learn

* How Claude differs from traditional search engines
* The mindset required to use AI effectively
* Techniques to improve prompts and outputs
* How to use AI for:

  * problem-solving
  * learning
  * ideation
  * structured thinking
